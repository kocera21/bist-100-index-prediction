# Documentation

This folder contains all project documentation, presentations, and related materials.

## Files

- **Report.pdf** - Comprehensive project report with detailed analysis, methodology, results, and conclusions about the BIST-100 prediction models
- **Presentation.pdf** - Project presentation slides in PDF format for easy viewing
- **Presentation.pptx** - Editable PowerPoint presentation with all slides and speaker notes
- **presentation_link.txt** - Link to the online presentation or additional resources

## Contents Summary

The documentation includes:
- Project objectives and scope
- Data sources and preprocessing methodology
- Model architecture and implementation details
- Training and validation procedures
- Performance metrics and comparative analysis
- Visualizations and forecasting results
- Conclusions and recommendations
- References and bibliography

## How to Use

1. **For a quick overview**: Start with `Presentation.pdf`
2. **For detailed analysis**: Read `Report.pdf`
3. **For presentations**: Use `Presentation.pptx` for editing or `Presentation.pdf` for viewing
4. **For additional resources**: Check `presentation_link.txt`

## Notes

All documentation is in English and covers the complete project from data collection through model evaluation.
